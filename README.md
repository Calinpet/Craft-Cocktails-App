# Craft-Cocktails-App

A simple webb app visualizing and random choosing cocktails recipes.

## Technologies
- HTML5
- CSS3
- JavaScript/jQuery
- [CocktailsDB API](https://www.thecocktaildb.com/api.php?ref=apilist.fun)
